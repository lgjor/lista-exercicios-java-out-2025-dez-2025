# X70: maxMirror

Vamos considerar que uma seção "espelho" em um array é um grupo de elementos contíguos tal que, em algum lugar do array, o mesmo grupo aparece em ordem inversa. Por exemplo, a maior seção espelho em {1, 2, 3, 8, 9, 3, 2, 1} tem comprimento 3 (a parte {1, 2, 3}). Retorne o tamanho da maior seção espelho encontrada no array fornecido.
