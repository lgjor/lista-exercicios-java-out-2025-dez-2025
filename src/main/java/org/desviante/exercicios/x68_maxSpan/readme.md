# X68: maxSpan

Considere as ocorrências mais à esquerda e mais à direita de um determinado valor em um array. Digamos que o intervalo (ou span) seja o número de elementos entre esses dois intervalos, inclusive. Um único valor tem um intervalo igual a 1. Retorne o comprimento do maior intervalo encontrado no array fornecido. (A eficiência não é uma prioridade.)
