# X69: fix34

Retorne um array que contenha exatamente os mesmos números do array fornecido, mas reorganizados de forma que cada 3 seja imediatamente seguido por um 4. Não mova os 3s, mas todos os outros números podem ser movidos. O array contém a mesma quantidade de 3s e 4s, cada 3 tem um número após ele que não é 3 nem 4, e um 3 aparece no array antes de qualquer 4.
