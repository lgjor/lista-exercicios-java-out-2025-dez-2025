# X54: fizzArray2

Dado um número `n`, crie e retorne um novo array de strings de comprimento `n`, contendo as strings `"0", "1", "2" ... até n-1`. `N` pode ser 0, caso em que basta retornar um array de comprimento 0. Observe que isso `String.valueOf(xxx)` criará o `String` formato da maioria dos tipos, incluindo `int`. A sintaxe para criar um novo array de strings é `new String[desired_length]`.
