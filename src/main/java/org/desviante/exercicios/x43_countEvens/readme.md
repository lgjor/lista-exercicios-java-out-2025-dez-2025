# x43_countEvens

Escreva uma função em Java que receba um `int` array como parâmetro e retorne o número de `ints` pares que ele contém. Observação: o operador `%` "mod" calcula o resto, por exemplo, `5 % 2` é 1.

Exemplos:

```
countEvens({2, 1, 2, 3, 4}) -> 3

countEvens({2, 2, 0}) -> 3

countEvens({1, 3, 5}) -> 0
```
